  - 로스터 노트: "블루베리, 초콜릿, 플로럴"
  - **개인 매칭 점수: 85/100점** ⭐⭐⭐⭐
  - **개인 강점 분석**: "블루베리 인지 excellent! 초콜릿 구분 전문가 수준"
- **AI 개인 코치**: 
  - "당신의 과일향 인지 능력이 상위 15%입니다"

  - "3개월 전과 비교해 정확도 23% 향상되었어요!"

  - "당신의 산지 구별 실력이 크게 향상되었어요!"
